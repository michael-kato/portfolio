Elecric Outline Effect (Blender)

I created an effect to transform any 3D model into a set of colorful dots connected by lighting

The geometry node setup in Blender. It starts by converting the mesh into a point cloud, connects the points to their closest neighbors with a spline, subdivides the spline and apply random noise to it, then extrude a cylinder along the path for the visual geo. A color ramp is applied along the Z axis and each point is assigned a color as an attribute. This color is later applied in the shader by extracting the attribute into the emissive property
